MovieRatings

Plateforme d’analyse des avis de films.

MovieRatings est une plateforme expérimentale d’analyse des avis de films. Ce site a été développé dans le cadre d’un projet universitaire en informatique et utilise des technologies d’intelligence artificielle pour analyser le sentiment exprimé dans les critiques de films.

Les avis sont récupérés automatiquement depuis l’API publique The Movie Database (TMDB), puis traités par un modèle de langage (BERT / TextBlob) afin de déterminer si le ton général est positif, négatif ou neutre.
L'application fonctionne comme suit :

L'utilisateur saisit le titre d’un film.
MovieRatings interroge l’API publique The Movie Database (TMDB) pour récupérer les avis associés à ce film.
Chaque avis est analysé automatiquement grâce à un modèle de langage (BERT/TextBlob), qui détermine si le ton général est positif, négatif ou neutre.

Les résultats de cette analyse sont enregistrés dans une base de données SQLite et présentés sous forme de statistiques et graphiques pour une visualisation claire.
