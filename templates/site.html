Le Projet

MovieRatings est une plateforme expérimentale d’analyse des avis de films. Ce site a été développé dans le cadre d’un projet universitaire en informatique et utilise des technologies d’intelligence artificielle pour analyser le sentiment exprimé dans les critiques de films.

Les avis sont récupérés automatiquement depuis l’API publique TMDB, puis traités par un modèle BERT afin de déterminer si le ton général est positif ou négatif.

Fonctionnement

1. L'utilisateur saisit le titre d’un film dans le Dashboard.
2. L’API The Movie Database (TMDB) est interrogée pour récupérer les avis.
3. Chaque avis est analysé par le modèle de langage BERT (5 niveaux de sentiments).

Les résultats sont enregistrés en base de données et présentés sous forme de statistiques claires.
